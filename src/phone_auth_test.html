PAYO — Firebase Phone Auth Test Page

This performs a REAL phone sign-in using Firebase — an actual SMS will be sent to the number you enter. Use your own real phone number for this test.

Step 1 — Fill in your Firebase config near the bottom of this file

Same config you used for the FCM test page (Project Settings → General → Your apps → web app). Phone Auth does not need a VAPID key.

Step 2 — Enter phone number (with country code) and send OTP

(status will appear here)

Step 3 — Enter the OTP you received via real SMS

Step 4 — Copy this ID token into Postman

Use it as the idToken value when calling POST /auth/verify-firebase-token.

(ID token will appear here after successful verification)
